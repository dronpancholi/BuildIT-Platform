## What Is This Really?

Project 31A is a **Vertical SaaS + Agentic Workflow Platform** for SEO link-building operations.

It is NOT primarily a "dashboard." It is NOT a thin API wrapper. It is NOT an AI chatbot for SEO advice.

It is a **durable workflow execution engine** with an SEO-domain intelligence layer bolted on top. The Temporal.io orchestration backbone makes it architecturally closer to an enterprise workflow automation product (like Camunda, Temporal Cloud, or Zapier for enterprises) than to a typical SEO SaaS tool.

---

## Classification

### Primary Category: **Vertical SaaS**
- Domain: SEO Operations / Link Building Automation
- Target: Agencies and enterprise SEO teams
- Business model: Seat-based or tenant-based monthly subscription

### Secondary Category: **Agentic Workflow Engine**
- Temporal.io backbone with 10 registered workflows
- LLM-powered activity execution with semantic validation
- Human-in-the-loop approval gates via workflow signals
- This is genuinely closer to an "AI agent system" than a conventional SaaS

### Tertiary Category: **AI-Augmented Outreach Platform**
- LLM email generation with semantic grounding validation
- Anti-fluff phrase enforcement (9 prohibited patterns)
- Multi-step follow-up sequence automation
- Tiered pitching (editorial DR<75 vs. data journalism DR≥75)

## What It Is Not

- **Not a "ChatGPT for SEO"** — the LLM is one component, not the core
- **Not a link exchange or PBN tool** — no black-hat functionality detected
- **Not a pure data product** — it doesn't sell data; it sells automation
- **Not a CRM** — it triggers CRM data ingestion but doesn't replace CRM
- **Not finished enterprise software** — no SSO, no audit log UI, no role-based access UI

---

## True Identity Statement

> **Project 31A is a durable agentic workflow platform for autonomous SEO link-building operations, built on Temporal.io, with LLM-powered personalized outreach and real HTTP link verification. It is technically pre-commercial but architecturally serious.**
